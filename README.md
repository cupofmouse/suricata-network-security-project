# Suricata 기반 네트워크 공격 탐지 및 자동 대응 시스템

## 프로젝트 소개

본 프로젝트는 Suricata IDS를 이용하여 네트워크 공격을 탐지하고,
Python을 활용하여 자동 대응 시스템을 구축하는 것을 목표로 한다.

프로젝트 기간

2026.06 ~ 2026.08

## 개발 환경

- Ubuntu Server
- Kali Linux
- Python
- Suricata IDS
